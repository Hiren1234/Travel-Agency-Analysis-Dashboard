# ✈️ Travel Agency Analytics Dashboard | Power BI

## 📌 Project Overview

The **Travel Agency Analytics Dashboard** is an end-to-end Business Intelligence solution developed using **Microsoft Power BI** to analyze travel business performance across sales, customers, operations, payments, visa processing, and customer satisfaction.

The dashboard transforms raw travel data into actionable insights that help business leaders make data-driven decisions, optimize revenue, improve customer experience, and enhance operational efficiency.

---

## 🎯 Objectives

- Analyze overall travel business performance
- Track revenue, bookings, and customer trends
- Identify high-performing travel packages
- Understand customer behavior and lifetime value
- Monitor payment and operational performance
- Evaluate visa processing efficiency
- Improve customer satisfaction using review analytics
